# GuruNetwork Automatic Mailer

GuruNetwork Automatic Mailer is a backend service that acts as a subcomponent of the GuruNetwork platform.

It is responsible for automatically sending a confirmation email to users after they submit an inquiry form on the GuruNetwork website. The email confirms that the inquiry has been received and informs the user that the team will review it and get back to them soon.

---

## 🎯 Purpose

- Provide immediate feedback to users after form submission
- Improve user trust and communication
- Eliminate the need for manual acknowledgement emails

---

## 🔁 Workflow

1. User fills out an inquiry form on the GuruNetwork website  
2. Form data is submitted to the backend  
3. Automatic Mailer is triggered  
4. A custom confirmation email is sent to the user  
5. User receives acknowledgement instantly  

---

## ✉️ Email Functionality

- Sends automatic acknowledgement emails
- Uses customizable email templates
- Confirms inquiry receipt
- Communicates next steps clearly to the user
